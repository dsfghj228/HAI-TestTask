# HAI-TestTask

Тестовое задание на позицию **Junior FullStack C#/.Net/React Developer**.  

Проект представляет собой небольшой медицинский сервис с функционалом:

- Получение списка врачей
- Регистрация нового врача
- Получение списка заболеваний
- Изменение данных о болезни
- Получение списка пациентов
- Получение отдельного пациента
- Создание новых пациентов
- Добавление болезни пациенту

---

## Используемые технологии

### Backend
- **ASP.NET Core** – для создания REST API
- **PostgreSQL** – база данных
- **Entity Framework Core** – ORM для работы с базой данных
- **MediatR** – для реализации паттерна CQRS
- **FluentValidation** – для валидации входных данных
- **AutoMapper** – для маппинга DTO и сущностей

### Frontend
- **React** с **TypeScript** – для создания клиентской части
- **Axios** – для работы с API
- **Tailwind CSS** – для стилизации интерфейса

---

## Запуск проекта

### Backend
   ```bash
   cd Back
   dotnet run
```
### Frontend
```bash
   cd front
   npm start
```
